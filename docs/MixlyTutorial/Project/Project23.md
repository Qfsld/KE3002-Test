# 项目二十三 红外发射

## 1.实验说明

在这个套件中，有一个红外发射传感器，它主要用到了红外发射管。它是一个能发射出特定波长红外光的二极管。我们可以将传感器连接到单片机上，利用编程，控制传感器发射出38KHz调制信号，可适应市面上各种红外接收头，以便红外线接收传感器能接收到，从而实现红外无线通讯。

实验中，我们利用红外发射传感器发射对应数据，每发射一次数据，传感器上的D1 LED就闪烁一次。
